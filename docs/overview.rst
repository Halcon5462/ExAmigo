Обзор проекта
=============

Этот проект реализует backend API сервиса подготовки к экзаменам.
Система включает:

- управление пользователями и авторизацией;
- модуль достижений и событийных стратегий;
- библиотеку задач с проверкой и оценкой решений;
- магазин внутриигровых товаров и косметики;
- систему статистики и пользовательских прогрессов;
- режимы поддержки подсказок и помощи на основе AI;
- websocket-компоненты для соревнований.

Документация строится через Sphinx и показывает структуру модулей, модели, сериализаторы, вьюшки и сервисы.

Сборку HTML-документации можно выполнить командой:

.. code-block:: bash

   cd docs
   make html

Готовая документация появляется в `docs/_build/html`.

CI/CD интеграция
-----------------

Документация автоматически собирается при каждом коммите в основную ветку через GitHub Actions.
Конфигурация находится в `.github/workflows/docs.yml`.

Для локальной сборки убедитесь, что установлены зависимости:

.. code-block:: bash

   pip install sphinx sphinx_rtd_theme

Затем выполните сборку:

.. code-block:: bash

   cd docs
   sphinx-build -b html . _build/html
