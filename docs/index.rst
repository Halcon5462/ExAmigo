Документация API сервиса экзаменов
===================================

Проект содержит backend API, реализующий сервис подготовки к экзаменам с достижениями, статистикой, магазинам и поддержкой подсказок.

Документация автоматически собирается Sphinx и описывает основные модули, классы и функции проекта.
