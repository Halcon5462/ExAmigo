Система оценивания
==================

Общее
-----

Баллы считаются на основе:
- правильных ответов
- сложности задания
- веса задания

Логика:
-------

1. Проверяются ответы пользователя
2. Считается количество правильных
3. Применяется коэффициент сложности
4. Итог записывается в ExamSession.score
