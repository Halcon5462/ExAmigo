Система streak
===============

Общее
-----

Streak — система ежедневной активности пользователя.

Логика:
-------

- пользователь решает задания → streak увеличивается
- пропуск дня → streak сбрасывается
- данные хранятся в модели Streak

Используется для:
-----------------

- мотивации
- наград
- прогресса
